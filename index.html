Como o QR passa a funcionar no celular
Geração

Ao criar a etiqueta, o sistema monta um link com os dados do item codificados para abrir uma tela mobile.

Leitura

Ao escanear no celular, o navegador abre uma página HTML leve com SKU, lote, validade, método e endereço.

Uso real

Para funcionar entre dispositivos, o arquivo precisa estar hospedado em um site, servidor local da escola ou GitHub Pages.

Em modo local no seu computador, o QR ainda consegue ser testado pelo botão de simulação. Para leitura real em outro aparelho, publique o arquivo em uma URL acessível.
